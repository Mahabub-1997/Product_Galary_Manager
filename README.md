# Product Gallery Manager

A Laravel-based application for managing products and categories with advanced features for admin and user interactions. The project supports multiple product images with AJAX-powered upload, update, and deletion functionalities.

---

## Features

### User
- Secure login and authentication
- Access to view available products and their details
- Update personal profile information (name, email, password,phone,image etc.)

### Admin
- Secure admin login
- Full control over categories and products
- Dashboard with quick stats:
  - Total users
  - Total categories
  - Total products
- Category management:
  - Create new categories
  - Edit and delete existing categories
- Product management under categories:
  - Select a category when creating a product
  - Upload multiple images per product
  - Update and remove images dynamically using AJAX
  - Edit, delete, and view products
- Admin can update own profile

### Product Images
- Multiple image upload per product
- AJAX-based operations:
  - Add new images
  - Remove existing images
  - Update images without page reload
